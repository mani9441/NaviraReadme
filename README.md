# NaviraReadme

1. **Backend (FastAPI + GitHub API + Gemini)** – handles fetching repo, generating drafts, managing actions, finalizing README.
2. **Frontend (React)** – interactive draft editor, Markdown preview, buttons for Accept / Retry / Skip.


---

## **1️⃣ Backend: FastAPI Skeleton**

Create a folder structure like:

```
navira/
│
├─ backend/
│   ├─ main.py
│   ├─ drafts.py
│   ├─ github_fetcher.py
│   ├─ gemini_client.py
│   └─ requirements.txt
```
